# 使用说明

## python

### python/ipinfo.py

-   功能：

    -   查询IP或域名信息

-   语法：
    ```
    ipinfo.py [IP|DOMAIN_NAME]
    ```

-   举例：

    ```
    ipinfo.py 8.8.8.8
    ipinfo.py baidu.com
    ```

### python/yd.py

-   功能：
    -   有道翻译
-   语法：

    ```
    yd.py [WORLD]
    ```

-   举例：

    ```
    yd.py love
    yd.py 中国
    yd.py 'I love china!'
    ```
